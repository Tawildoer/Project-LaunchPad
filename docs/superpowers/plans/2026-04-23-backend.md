# Ground Station Backend Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the ground station Python backend — a FastAPI server that bridges MAVLink telemetry from the drone to browser clients via WebSocket, brokers WebRTC signaling, serves the frontend, hosts the Claude copilot, and monitors the mission queue.

**Architecture:** A single FastAPI application wired together in `main.py`. An asyncio background task maintains a persistent TCP connection to `drone-agent` on the Tailscale network, parsing incoming MAVLink frames and pushing the decoded telemetry to all connected browser WebSocket clients at 10 Hz. Commands flow in reverse: the WebSocket handler receives validated command JSON from the browser and serialises it back to MAVLink for the drone. The Claude copilot module is a standalone async function called by a FastAPI endpoint; it holds per-session conversation history in memory and applies a risk policy before returning responses.

**Tech Stack:** Python 3.11+, FastAPI, uvicorn, pymavlink, anthropic SDK, jsonschema, httpx (for Nominatim geocoding), pytest, pytest-asyncio

```python
"""
Schema loader — reads shared JSON Schema files once at import time.
Provides typed validate_* helpers used across the backend.
"""
from __future__ import annotations

import json
from pathlib import Path
from typing import Any

import jsonschema
from jsonschema import Draft7Validator

_SCHEMAS_DIR = Path(__file__).parents[3] / "shared" / "schemas"


def _load(name: str) -> dict[str, Any]:
    path = _SCHEMAS_DIR / f"{name}.json"
    with path.open() as f:
        return json.load(f)


_TELEMETRY_SCHEMA = _load("telemetry")
_COMMAND_SCHEMA = _load("commands")
_WEBSOCKET_SCHEMA = _load("websocket")

_TELEMETRY_VALIDATOR = Draft7Validator(_TELEMETRY_SCHEMA)
_COMMAND_VALIDATOR = Draft7Validator(_COMMAND_SCHEMA)
_WEBSOCKET_VALIDATOR = Draft7Validator(_WEBSOCKET_SCHEMA)


class SchemaValidationError(ValueError):
    """Raised when a message fails JSON Schema validation."""


def validate_telemetry(data: dict[str, Any]) -> None:
    errors = list(_TELEMETRY_VALIDATOR.iter_errors(data))
    if errors:
        msg = "; ".join(e.message for e in errors[:3])
        raise SchemaValidationError(f"Invalid telemetry: {msg}")


def validate_command(data: dict[str, Any]) -> None:
    errors = list(_COMMAND_VALIDATOR.iter_errors(data))
    if errors:
        msg = "; ".join(e.message for e in errors[:3])
        raise SchemaValidationError(f"Invalid command: {msg}")


def validate_ws_envelope(data: dict[str, Any]) -> None:
    errors = list(_WEBSOCKET_VALIDATOR.iter_errors(data))
    if errors:
        msg = "; ".join(e.message for e in errors[:3])
        raise SchemaValidationError(f"Invalid WebSocket envelope: {msg}")
```

```python
"""
MAVLink receiver — connects to drone-agent over TCP, parses MAVLink frames,
and maintains a shared TelemetryState updated on each relevant message.

Environment variables:
  DRONE_HOST  — Tailscale IP of drone-agent (default: 100.64.0.1)
  DRONE_PORT  — TCP port for MAVLink stream   (default: 5760)
"""
from __future__ import annotations

import asyncio
import logging
import math
import os
import time
from dataclasses import dataclass, field
from typing import Any

from pymavlink import mavutil

logger = logging.getLogger(__name__)

DRONE_HOST = os.environ.get("DRONE_HOST", "100.64.0.1")
DRONE_PORT = int(os.environ.get("DRONE_PORT", "5760"))

# MAVLink base_mode bit 7: armed
_MAV_MODE_FLAG_SAFETY_ARMED = 128

# Mapping from ArduPilot custom_mode integers to human-readable mode strings.
# Values match ArduPlane + QuadPlane mode numbers.
_ARDUPLANE_MODES: dict[int, str] = {
    0:  "MANUAL",
    1:  "CIRCLE",
    2:  "STABILIZE",
    3:  "TRAINING",
    4:  "ACRO",
    5:  "FBWA",
    6:  "FBWB",
    7:  "CRUISE",
    8:  "AUTOTUNE",
    10: "AUTO",
    11: "RTL",
    12: "LOITER",
    13: "TAKEOFF",
    15: "GUIDED",
    17: "QSTABILIZE",
    18: "QHOVER",
    19: "QLOITER",
    20: "QLAND",
    21: "QRTL",
    22: "QAUTOTUNE",
    23: "QACRO",
    24: "THERMAL",
}


def resolve_mode(custom_mode: int) -> str:
    """Return a human-readable mode string for an ArduPilot custom_mode integer."""
    return _ARDUPLANE_MODES.get(custom_mode, f"MODE_{custom_mode}")


@dataclass
class TelemetryState:
    """Shared mutable state updated by build_telemetry_dict and read by the broadcast loop."""

    timestamp: float = field(default_factory=lambda: time.time() * 1000)
    armed: bool = False
    mode: str = "UNKNOWN"
    position: dict[str, float] = field(default_factory=lambda: {
        "lat": 0.0, "lon": 0.0, "alt_msl": 0.0, "alt_rel": 0.0
    })
    attitude: dict[str, float] = field(default_factory=lambda: {
        "roll": 0.0, "pitch": 0.0, "yaw": 0.0
    })
    velocity: dict[str, float] = field(default_factory=lambda: {
        "ground_speed": 0.0, "air_speed": 0.0, "climb_rate": 0.0
    })
    battery: dict[str, Any] = field(default_factory=lambda: {
        "voltage": 0.0, "current": 0.0, "remaining_pct": 0
    })
    gps: dict[str, int] = field(default_factory=lambda: {
        "fix_type": 0, "satellites_visible": 0
    })
    mission_current: int = 0
    mission_total: int = 0
    active_poi_id: str | None = None

    def to_dict(self) -> dict[str, Any]:
        """Serialise to the shared telemetry schema dict."""
        d: dict[str, Any] = {
            "timestamp": self.timestamp,
            "armed": self.armed,
            "mode": self.mode,
            "position": dict(self.position),
            "attitude": dict(self.attitude),
            "velocity": dict(self.velocity),
            "battery": dict(self.battery),
            "gps": dict(self.gps),
        }
        if self.mode == "AUTO":
            d["mission"] = {
                "current_item": self.mission_current,
                "total_items": self.mission_total,
                "active_poi_id": self.active_poi_id,
            }
        return d


def build_telemetry_dict(state: TelemetryState, msg: Any) -> None:
    """
    Update *state* in-place from a single MAVLink message.
    Only processes message types relevant to the telemetry schema.
    All other message types are silently ignored.
    """
    msg_type = msg.get_type()

    if msg_type == "HEARTBEAT":
        state.armed = bool(msg.base_mode & _MAV_MODE_FLAG_SAFETY_ARMED)
        state.mode = resolve_mode(msg.custom_mode)
        state.timestamp = time.time() * 1000

    elif msg_type == "GLOBAL_POSITION_INT":
        state.position = {
            "lat":     msg.lat  / 1e7,
            "lon":     msg.lon  / 1e7,
            "alt_msl": msg.alt  / 1e3,
            "alt_rel": msg.relative_alt / 1e3,
        }
        # Ground speed from vx/vy (cm/s → m/s)
        vx = msg.vx / 100.0
        vy = msg.vy / 100.0
        vz = msg.vz / 100.0
        state.velocity["ground_speed"] = math.sqrt(vx**2 + vy**2)
        state.velocity["climb_rate"]   = -vz  # MAVLink vz positive = down
        state.timestamp = time.time() * 1000

    elif msg_type == "ATTITUDE":
        state.attitude = {
            "roll":  math.degrees(msg.roll),
            "pitch": math.degrees(msg.pitch),
            "yaw":   math.degrees(msg.yaw) % 360,
        }

    elif msg_type == "VFR_HUD":
        state.velocity["air_speed"] = msg.airspeed
        state.velocity["ground_speed"] = msg.groundspeed
        state.velocity["climb_rate"]   = msg.climb

    elif msg_type == "SYS_STATUS":
        state.battery = {
            "voltage":       msg.voltage_battery / 1000.0,
            "current":       msg.current_battery / 100.0,
            "remaining_pct": int(msg.battery_remaining),
        }

    elif msg_type == "GPS_RAW_INT":
        state.gps = {
            "fix_type":          int(msg.fix_type),
            "satellites_visible": int(msg.satellites_visible),
        }

    elif msg_type == "MISSION_CURRENT":
        state.mission_current = int(msg.seq)

    elif msg_type == "MISSION_COUNT":
        state.mission_total = int(msg.count)


class MavlinkReceiver:
    """
    Async TCP client that connects to drone-agent, parses MAVLink frames,
    and continuously updates the shared TelemetryState.

    Usage::

        state = TelemetryState()
        receiver = MavlinkReceiver(state)
        asyncio.create_task(receiver.run())
    """

    def __init__(
        self,
        state: TelemetryState,
        host: str = DRONE_HOST,
        port: int = DRONE_PORT,
        reconnect_delay: float = 5.0,
    ) -> None:
        self.state = state
        self.host = host
        self.port = port
        self.reconnect_delay = reconnect_delay
        self._connected = False

    @property
    def connected(self) -> bool:
        return self._connected

    async def run(self) -> None:
        """
        Connect to drone-agent and parse MAVLink forever.
        On connection loss, waits reconnect_delay seconds then retries.
        This coroutine never returns — run it as a background task.
        """
        while True:
            try:
                logger.info("Connecting to drone-agent at %s:%d", self.host, self.port)
                reader, _ = await asyncio.open_connection(self.host, self.port)
                self._connected = True
                logger.info("Connected to drone-agent")

                # pymavlink works with a file-like object; we feed it raw bytes
                # via a byte buffer rather than opening a real socket, so we
                # read from the asyncio stream and push bytes to a mavutil connection.
                #
                # Use mavutil.mavlink_connection with input_logfile for byte-by-byte
                # feeding is unreliable; instead, read chunks and parse with
                # MAVLink().decode() directly.
                mav = mavutil.mavlink_connection(
                    f"tcp:{self.host}:{self.port}",
                    source_system=255,
                    input=False,  # we drive reads manually
                )

                while True:
                    chunk = await reader.read(4096)
                    if not chunk:
                        raise ConnectionResetError("drone-agent closed connection")
                    mav.buf += chunk
                    while True:
                        msg = mav.parse_char(b"")
                        if msg is None:
                            break
                        build_telemetry_dict(self.state, msg)

            except (ConnectionRefusedError, ConnectionResetError, OSError) as exc:
                self._connected = False
                logger.warning("MAVLink connection lost: %s. Retrying in %.0fs", exc, self.reconnect_delay)
                await asyncio.sleep(self.reconnect_delay)
```

> **Implementation note:** pymavlink's `mavlink_connection("tcp:host:port")` opens a blocking socket internally. In the asyncio context we read from the asyncio stream ourselves and feed bytes to pymavlink's buffer manually. The `parse_char(b"")` call drains the internal buffer until no complete frame is available. This keeps the event loop unblocked. See also: the integration task (Task 9) for the final wiring.

```python
"""
WebSocket client registry and 10 Hz broadcast loop.

ClientRegistry tracks all connected browser WebSocket connections.
build_telemetry_envelope serialises TelemetryState to the websocket.json envelope.
The broadcast loop is started as an asyncio background task in main.py.
"""
from __future__ import annotations

import asyncio
import json
import logging
import time
from typing import Any

from fastapi import WebSocket

from src.mavlink_receiver import TelemetryState

logger = logging.getLogger(__name__)

BROADCAST_HZ = 10
BROADCAST_INTERVAL = 1.0 / BROADCAST_HZ


class ClientRegistry:
    """Thread-safe registry of connected browser WebSocket clients."""

    def __init__(self) -> None:
        self._clients: set[WebSocket] = set()
        self._lock = asyncio.Lock()

    @property
    def count(self) -> int:
        return len(self._clients)

    async def add(self, websocket: WebSocket) -> None:
        async with self._lock:
            self._clients.add(websocket)
        logger.info("Client connected. Total: %d", self.count)

    async def remove(self, websocket: WebSocket) -> None:
        async with self._lock:
            self._clients.discard(websocket)
        logger.info("Client disconnected. Total: %d", self.count)

    async def broadcast(self, message: str) -> None:
        """Send *message* to all clients. Silently removes any that fail."""
        dead: list[WebSocket] = []
        async with self._lock:
            clients = set(self._clients)

        for client in clients:
            try:
                await client.send_text(message)
            except Exception:
                dead.append(client)

        for client in dead:
            await self.remove(client)


def build_telemetry_envelope(state: TelemetryState) -> str:
    """Serialise TelemetryState to a JSON WebSocket envelope string."""
    envelope: dict[str, Any] = {
        "type": "telemetry",
        "payload": state.to_dict(),
    }
    return json.dumps(envelope)


async def broadcast_loop(state: TelemetryState, registry: ClientRegistry) -> None:
    """
    Runs forever at BROADCAST_HZ, pushing telemetry to all connected clients.
    Start as: asyncio.create_task(broadcast_loop(state, registry))
    """
    while True:
        tick_start = time.monotonic()
        if registry.count > 0:
            message = build_telemetry_envelope(state)
            await registry.broadcast(message)
        elapsed = time.monotonic() - tick_start
        await asyncio.sleep(max(0.0, BROADCAST_INTERVAL - elapsed))
```

```python
"""
Command handler — converts validated command dicts (shared/schemas/commands.json)
to MAVLink message dicts and enqueues them for transmission to drone-agent.

The queue consumer in MavlinkReceiver.run() reads from mav_send_queue and
serialises each dict to a real MAVLink frame over the TCP connection.

MAVLink command references:
  MAV_CMD_COMPONENT_ARM_DISARM = 400
  MAV_CMD_NAV_RETURN_TO_LAUNCH = 20
  SET_MODE base_mode field: 1 = MAV_MODE_FLAG_CUSTOM_MODE_ENABLED
"""
from __future__ import annotations

import asyncio
import logging
from typing import Any

logger = logging.getLogger(__name__)

# MAVLink command IDs (protocol constants — not hardware-specific)
_MAV_CMD_ARM_DISARM         = 400
_MAV_CMD_RTL                = 20
_MAV_MODE_FLAG_CUSTOM_MODE  = 1

# ArduPlane custom_mode numbers (matches _ARDUPLANE_MODES in mavlink_receiver.py)
_MODE_NUMBERS: dict[str, int] = {
    "MANUAL":     0,
    "STABILIZE":  2,
    "FBWA":       5,
    "AUTO":       10,
    "RTL":        11,
    "LOITER":     12,
    "GUIDED":     15,
    "QSTABILIZE": 17,
    "QHOVER":     18,
    "QLOITER":    19,
    "QLAND":      20,
    "QRTL":       21,
}

# MAV_FRAME_GLOBAL_RELATIVE_ALT_INT = 6
_MAV_FRAME_GLOBAL_RELATIVE_ALT_INT = 6

# Type mask: ignore velocity/accel/yaw (only position matters for fly_to)
_POSITION_ONLY_TYPE_MASK = 0b111111111000


class CommandHandler:
    """
    Converts command dicts to MAVLink message dicts and enqueues them.

    Each dict in the queue has a "mav_command" key identifying the
    MAVLink message type, plus the fields required for that type.
    The actual bytes-level serialisation happens in the send path of
    MavlinkReceiver (Task 9 wiring).
    """

    def __init__(self, mav_send_queue: asyncio.Queue) -> None:
        self._queue = mav_send_queue

    async def handle(self, command: dict[str, Any]) -> None:
        """Route a validated command dict to the appropriate MAVLink messages."""
        cmd_type = command.get("type")

        if cmd_type == "arm":
            await self._queue.put({
                "mav_command": "COMMAND_LONG",
                "command":     _MAV_CMD_ARM_DISARM,
                "param1":      1.0,
                "param2":      0.0,
                "param3":      0.0,
                "param4":      0.0,
                "param5":      0.0,
                "param6":      0.0,
                "param7":      0.0,
            })

        elif cmd_type == "disarm":
            await self._queue.put({
                "mav_command": "COMMAND_LONG",
                "command":     _MAV_CMD_ARM_DISARM,
                "param1":      0.0,
                "param2":      0.0,
                "param3":      0.0,
                "param4":      0.0,
                "param5":      0.0,
                "param6":      0.0,
                "param7":      0.0,
            })

        elif cmd_type == "return_home":
            await self._queue.put({
                "mav_command": "COMMAND_LONG",
                "command":     _MAV_CMD_RTL,
                "param1":      0.0,
                "param2":      0.0,
                "param3":      0.0,
                "param4":      0.0,
                "param5":      0.0,
                "param6":      0.0,
                "param7":      0.0,
            })

        elif cmd_type == "set_mode":
            mode_name = command["mode"]
            await self._queue.put({
                "mav_command": "SET_MODE",
                "base_mode":   _MAV_MODE_FLAG_CUSTOM_MODE,
                "custom_mode": _MODE_NUMBERS.get(mode_name, 0),
                "mode_name":   mode_name,
            })

        elif cmd_type == "fly_to":
            # Step 1: switch to GUIDED so the FC accepts position targets
            await self._queue.put({
                "mav_command": "SET_MODE",
                "base_mode":   _MAV_MODE_FLAG_CUSTOM_MODE,
                "custom_mode": _MODE_NUMBERS["GUIDED"],
                "mode_name":   "GUIDED",
            })
            # Step 2: send the position target
            await self._queue.put({
                "mav_command": "SET_POSITION_TARGET_GLOBAL_INT",
                "time_boot_ms": 0,
                "target_system": 1,
                "target_component": 1,
                "coordinate_frame": _MAV_FRAME_GLOBAL_RELATIVE_ALT_INT,
                "type_mask":  _POSITION_ONLY_TYPE_MASK,
                "lat_int":    int(command["lat"] * 1e7),
                "lon_int":    int(command["lon"] * 1e7),
                "alt":        float(command["alt"]),
                "vx": 0.0, "vy": 0.0, "vz": 0.0,
                "afx": 0.0, "afy": 0.0, "afz": 0.0,
                "yaw": 0.0, "yaw_rate": 0.0,
            })

        elif cmd_type == "send_mission":
            pois: list[dict[str, Any]] = command["pois"]
            waypoints = _build_waypoints(pois)
            await self._queue.put({
                "mav_command": "MISSION_COUNT",
                "count": len(waypoints),
                "target_system": 1,
                "target_component": 1,
            })
            for seq, wp in enumerate(waypoints):
                await self._queue.put({**wp, "seq": seq})

        elif cmd_type == "set_idle_mode":
            mode_name = command["idle_mode"]
            await self._queue.put({
                "mav_command": "SET_MODE",
                "base_mode":   _MAV_MODE_FLAG_CUSTOM_MODE,
                "custom_mode": _MODE_NUMBERS.get(mode_name, 0),
                "mode_name":   mode_name,
            })

        else:
            raise ValueError(f"Unknown command type: {cmd_type!r}")


def _build_waypoints(pois: list[dict[str, Any]]) -> list[dict[str, Any]]:
    """
    Convert a list of POI dicts to a flat list of MAVLink MISSION_ITEM_INT dicts.

    For each POI we emit:
      1. NAV_WAYPOINT to fly to the POI location
      2. NAV_LOITER_TIME to loiter for dwell_seconds at the POI
      3. DO_SET_ROI to lock the gimbal on the POI centre

    MAVLink command IDs:
      NAV_WAYPOINT      = 16
      NAV_LOITER_TIME   = 19
      DO_SET_ROI        = 201
    """
    _NAV_WAYPOINT    = 16
    _NAV_LOITER_TIME = 19
    _DO_SET_ROI      = 201
    _MAV_FRAME_GLOBAL_RELATIVE_ALT = 3

    items: list[dict[str, Any]] = []

    for poi in pois:
        lat = poi["lat"]
        lon = poi["lon"]
        alt = poi.get("alt", 80.0)
        radius = poi["loiter_radius"]
        dwell  = poi["dwell_seconds"]

        # Fly to POI location
        items.append({
            "mav_command":       "MISSION_ITEM_INT",
            "command":           _NAV_WAYPOINT,
            "frame":             _MAV_FRAME_GLOBAL_RELATIVE_ALT,
            "param1":            0.0,     # hold time (s)
            "param2":            5.0,     # acceptance radius (m)
            "param3":            0.0,     # pass-through
            "param4":            float("nan"),  # yaw (NaN = auto)
            "x":                 int(lat * 1e7),
            "y":                 int(lon * 1e7),
            "z":                 float(alt),
            "target_system":     1,
            "target_component":  1,
            "current":           0,
            "autocontinue":      1,
        })

        # Lock gimbal on POI centre
        items.append({
            "mav_command":       "MISSION_ITEM_INT",
            "command":           _DO_SET_ROI,
            "frame":             _MAV_FRAME_GLOBAL_RELATIVE_ALT,
            "param1":            0.0,
            "param2":            0.0,
            "param3":            0.0,
            "param4":            0.0,
            "x":                 int(lat * 1e7),
            "y":                 int(lon * 1e7),
            "z":                 float(alt),
            "target_system":     1,
            "target_component":  1,
            "current":           0,
            "autocontinue":      1,
        })

        # Loiter at radius for dwell time
        items.append({
            "mav_command":       "MISSION_ITEM_INT",
            "command":           _NAV_LOITER_TIME,
            "frame":             _MAV_FRAME_GLOBAL_RELATIVE_ALT,
            "param1":            float(dwell),    # dwell time (s)
            "param2":            0.0,
            "param3":            float(radius),   # loiter radius (m, positive = CW)
            "param4":            0.0,
            "x":                 int(lat * 1e7),
            "y":                 int(lon * 1e7),
            "z":                 float(alt),
            "target_system":     1,
            "target_component":  1,
            "current":           0,
            "autocontinue":      1,
        })

    return items
```

```python
WebRTC signaling relay — forwards offer/answer/ICE between the browser
and drone-agent with minimal overhead.

# PACKAGE_BOUNDARY: packages/webrtc-signaling
# When packages/webrtc-signaling is built, replace this module with:

# The public API (from_browser, from_drone) remains identical.

Design constraint: video latency is the primary optimisation target.

and re-serialise to the other endpoint. No buffering, no state beyond the
drone WebSocket connection reference.

Environment variables:
  DRONE_SIGNAL_WS_URL — WebSocket URL of drone-agent's signaling endpoint
                        (default: ws://100.64.0.1:5761/signal)

import json
import logging
import os
from typing import Any

logger = logging.getLogger(__name__)

DRONE_SIGNAL_WS_URL = os.environ.get(
    "DRONE_SIGNAL_WS_URL", "ws://100.64.0.1:5761/signal"
)

_VALID_SIGNAL_TYPES = {"offer", "answer", "ice_candidate"}


class SignalingRelay:
    """
    Forwards WebRTC signaling messages between a browser WebSocket client
    and the drone-agent signaling WebSocket.

    Instantiate once per FastAPI application. The drone_ws is the persistent
    WebSocket connection to drone-agent's signaling endpoint. In tests, pass
    a mock directly via the drone_ws parameter.
    """

    def __init__(
        self,
        drone_ws: Any = None,
        drone_ws_factory: Any = None,
    ) -> None:
        # drone_ws: a connected WebSocket-like object with send_text(str) method.
        # drone_ws_factory: async callable that returns a drone_ws (used in production
        #                   to lazily connect; not used in unit tests).
        self._drone_ws = drone_ws
        self._factory = drone_ws_factory

    async def from_browser(self, browser_ws: Any, payload: dict[str, Any]) -> None:
        """
        Called when the browser sends a video_signal message.
        Forwards the signal to drone-agent.
        """
        signal_type = payload.get("signal_type")
        if signal_type not in _VALID_SIGNAL_TYPES:
            raise ValueError(f"Unknown signal_type: {signal_type!r}")

        envelope = json.dumps({
            "type": "video_signal",
            "payload": {
                "signal_type": signal_type,
                "data": payload["data"],
            },
        })
        logger.debug("Forwarding %s to drone", signal_type)
        await self._drone_ws.send_text(envelope)

    async def from_drone(self, browser_ws: Any, payload: dict[str, Any]) -> None:
        """
        Called when drone-agent sends a video_signal message.
        Forwards the signal to the browser.
        """
        signal_type = payload.get("signal_type")
        if signal_type not in _VALID_SIGNAL_TYPES:
            raise ValueError(f"Unknown signal_type: {signal_type!r}")

        envelope = json.dumps({
            "type": "video_signal",
            "payload": {
                "signal_type": signal_type,
                "data": payload["data"],
            },
        })
        logger.debug("Forwarding %s to browser", signal_type)
        await browser_ws.send_text(envelope)
```

```python
_MOVEMENT_MODES = {"RTL", "AUTO", "GUIDED"}

def classify_tool_call(tool_name: str, **tool_input: Any) -> ToolRisk:
    """

        return ToolRisk.REQUIRES_CONFIRMATION

    return ToolRisk.REQUIRES_CONFIRMATION

async def _execute_geocode(_state: TelemetryState, tool_input: dict) -> str:
    """
    Resolve a place name to lat/lon using OpenStreetMap Nominatim (free, no API key).
    Returns a JSON string: {"lat": float, "lon": float, "display_name": str}
    """
    place = tool_input["place_name"]
    url = "https://nominatim.openstreetmap.org/search"
    params = {"q": place, "format": "json", "limit": "1"}
    headers = {"User-Agent": "ProjectLaunchPad/1.0 (matt@specter.co)"}

    async with httpx.AsyncClient(timeout=8.0) as client:
        resp = await client.get(url, params=params, headers=headers)
        resp.raise_for_status()
        results = resp.json()

    if not results:
        return json.dumps({"error": f"No results found for '{place}'"})

    hit = results[0]
    return json.dumps({
        "lat": float(hit["lat"]),
        "lon": float(hit["lon"]),
        "display_name": hit["display_name"],
    })

def _tool_to_command(tool_name: str, tool_input: dict) -> dict[str, Any]:
    """
    Convert a Claude tool call into the command dict format from commands.json.
    Used for REQUIRES_CONFIRMATION tools — the dict is sent to the browser
    and, if confirmed, forwarded to CommandHandler.
    """
    if tool_name == "fly_to":
        return {"type": "fly_to", "lat": tool_input["lat"], "lon": tool_input["lon"], "alt": tool_input["alt"]}
    if tool_name == "return_home":
        return {"type": "return_home"}
    if tool_name == "set_mode":
        return {"type": "set_mode", "mode": tool_input["mode"]}
    if tool_name == "send_mission":
        return {"type": "send_mission", "pois": []}  # POIs come from MissionContext in browser
    if tool_name == "add_poi":
        import time
        poi_id = f"POI-{int(time.time()) % 10000:04d}"
        return {
            "type": "add_poi",
            "poi": {

                "alt": 80.0,

# ---------------------------------------------------------------------------
# CopilotSession
# ---------------------------------------------------------------------------

class CopilotSession:
    """
    Manages per-session conversation history and processes copilot requests.

    One instance per FastAPI application. Session history is keyed by session_id
    (a UUID generated in the browser, stable per tab, cleared on reload).
    """

    def __init__(
        self,
        state: TelemetryState,
        client: AsyncAnthropic | None = None,
    ) -> None:
        self._state = state
        self._client = client or AsyncAnthropic()
        # session_id → list of MessageParam dicts
        self._history: dict[str, list[MessageParam]] = {}

    async def chat(self, session_id: str, text: str) -> dict[str, Any]:
        """
        Process a natural language message and return a copilot_response payload dict.

        Returns one of:
          {"response_type": "answer", "text": str}
          {"response_type": "clarification", "text": str}
          {"response_type": "confirmation_required", "text": str, "pending_command": dict}
        """
        history = self._history.setdefault(session_id, [])
        history.append({"role": "user", "content": text})

        system = build_system_prompt(self._state)

        # First API call
        response = await self._client.messages.create(
            model=COPILOT_MODEL,
            max_tokens=1024,
            system=system,
            tools=_TOOLS,
            messages=history,
        )

        # Handle tool use
        if response.stop_reason == "tool_use":
            tool_block = next(b for b in response.content if b.type == "tool_use")
            tool_name  = tool_block.name
            tool_input = tool_block.input
            risk       = classify_tool_call(tool_name, **tool_input)

            if risk == ToolRisk.SAFE:
                # Auto-execute and feed result back to Claude
                history.append({"role": "assistant", "content": response.content})

                if tool_name == "get_drone_status":
                    result_str = await _execute_get_drone_status(self._state, tool_input)
                elif tool_name == "geocode":
                    result_str = await _execute_geocode(self._state, tool_input)
                else:
                    result_str = json.dumps({"error": f"No executor for {tool_name}"})

                history.append({
                    "role": "user",
                    "content": [{
                        "type": "tool_result",
                        "tool_use_id": tool_block.id,
                        "content": result_str,
                    }],
                })

                # Second API call with tool result
                follow_up = await self._client.messages.create(
                    model=COPILOT_MODEL,
                    max_tokens=1024,
                    system=system,
                    tools=_TOOLS,
                    messages=history,
                )
                answer_text = _extract_text(follow_up)
                history.append({"role": "assistant", "content": follow_up.content})
                return {"response_type": "answer", "text": answer_text}

            else:  # REQUIRES_CONFIRMATION
                command = _tool_to_command(tool_name, tool_input)
                description = _describe_command(tool_name, tool_input)
                # Do not add to history yet — the operator hasn't confirmed
                return {
                    "response_type": "confirmation_required",
                    "text": f"I'm going to: {description}. Please confirm or cancel.",
                    "pending_command": command,
                }

        # Plain text response (clarification question or direct answer)
        answer_text = _extract_text(response)
        history.append({"role": "assistant", "content": response.content})

        # Heuristic: if it ends with "?" it's a clarification, otherwise an answer
        response_type = "clarification" if answer_text.rstrip().endswith("?") else "answer"
        return {"response_type": response_type, "text": answer_text}

    def clear_session(self, session_id: str) -> None:
        """Discard conversation history for a session."""
        self._history.pop(session_id, None)


def _extract_text(response: Any) -> str:
    """Extract the first text block from an Anthropic message response."""
    for block in response.content:
        if block.type == "text":
            return block.text
    return ""


def _describe_command(tool_name: str, tool_input: dict) -> str:
    """Return a human-readable description of a tool call for the confirmation prompt."""
    if tool_name == "fly_to":
        return (
            f"fly to lat={tool_input['lat']:.5f}, lon={tool_input['lon']:.5f} "
            f"at {tool_input['alt']:.0f}m altitude"
        )
```

```python
"""
Mission monitor — watches TelemetryState and sends the configured idle mode
command when the mission queue empties while in AUTO mode.

Also broadcasts mission_update WebSocket envelopes on state changes.

The monitor is called from the broadcast loop background task (main.py) on
every telemetry tick, so it must be fast and side-effect-free if nothing changed.
"""
from __future__ import annotations

import asyncio
import json
import logging
from typing import Any

from src.mavlink_receiver import TelemetryState

logger = logging.getLogger(__name__)

_MAV_MODE_FLAG_CUSTOM_MODE = 1

_MODE_NUMBERS: dict[str, int] = {
    "LOITER":  12,
    "QLOITER": 19,
}


class MissionMonitor:
    """
    Detects mission completion and triggers idle mode.

    idle_mode: "LOITER" or "QLOITER" — updated by the operator via SettingsPanel.
    """

    def __init__(
        self,
        mav_send_queue: asyncio.Queue,
        client_registry: Any,  # ClientRegistry from websocket_server.py
        idle_mode: str = "LOITER",
    ) -> None:
        self._queue     = mav_send_queue
        self._registry  = client_registry
        self.idle_mode  = idle_mode

        # Track whether we already triggered idle for this mission
        self._idle_sent = False
        # Track last known total so we can detect a new non-empty mission
        self._last_total: int = 0

    async def check(self, state: TelemetryState) -> None:
        """
        Called on every telemetry tick. Checks for mission completion.
        Is idempotent — calling it twice with the same completed state
        does not send idle mode a second time.
        """
        if state.mode != "AUTO":
            # Not in AUTO — reset so a future mission can trigger idle
            self._idle_sent = False
            return

        total   = state.mission_total
        current = state.mission_current

        if total == 0:
            # No mission loaded
            return

        # Track the most recent non-zero total for re-trigger detection
        if total > 0:
            self._last_total = total

        mission_complete = (current >= total) and (self._last_total > 0)

        if mission_complete and not self._idle_sent:
            self._idle_sent = True
            logger.info(
                "Mission complete (%d/%d). Sending idle mode: %s",
                current, total, self.idle_mode,
            )
            await self._send_idle_mode()
            await self._broadcast_complete(state)

    async def _send_idle_mode(self) -> None:
        await self._queue.put({
            "mav_command": "SET_MODE",
            "base_mode":   _MAV_MODE_FLAG_CUSTOM_MODE,
            "custom_mode": _MODE_NUMBERS.get(self.idle_mode, 12),
            "mode_name":   self.idle_mode,
        })

    async def _broadcast_complete(self, state: TelemetryState) -> None:
        envelope: dict[str, Any] = {
            "type": "mission_update",
            "payload": {
                "pois":              [],
                "active_poi_index":  None,
                "status":            "complete",
            },
        }
        await self._registry.broadcast(json.dumps(envelope))
```

```python
@asynccontextmanager
async def lifespan(app: FastAPI) -> AsyncGenerator[None, None]:
    """
    FastAPI lifespan context manager.
    Creates all singletons and starts background tasks on startup.
    Cancels background tasks on shutdown.
    """
    global _telemetry_state, _client_registry, _command_handler
    global _mission_monitor, _copilot_session, _signaling_relay, _mav_send_queue

    _telemetry_state = TelemetryState()
    _mav_send_queue  = asyncio.Queue()
    _client_registry = ClientRegistry()
    _command_handler = CommandHandler(_mav_send_queue)
    _mission_monitor = MissionMonitor(_mav_send_queue, _client_registry, idle_mode="LOITER")
    _copilot_session = CopilotSession(state=_telemetry_state)
    _signaling_relay = SignalingRelay(drone_ws=None)  # drone WS connected lazily

    receiver = MavlinkReceiver(
        state=_telemetry_state,
        host=os.environ.get("DRONE_HOST", "100.64.0.1"),
        port=int(os.environ.get("DRONE_PORT", "5760")),
    )

    tasks = [
        asyncio.create_task(receiver.run(), name="mavlink-receiver"),
        asyncio.create_task(
            _broadcast_and_monitor_loop(_telemetry_state, _client_registry, _mission_monitor),
            name="broadcast-loop",
        ),
        asyncio.create_task(
            _drain_send_queue(receiver, _mav_send_queue),
            name="mav-send-queue",
        ),
    ]

    logger.info("Backend started. Connecting to drone-agent…")
    try:
        yield
    finally:
        for t in tasks:
            t.cancel()
        await asyncio.gather(*tasks, return_exceptions=True)
        logger.info("Backend shutdown complete.")


async def _broadcast_and_monitor_loop(
    state: TelemetryState,
    registry: ClientRegistry,
    monitor: MissionMonitor,
) -> None:
    """
    Combined 10 Hz loop: broadcasts telemetry and checks mission state.
    Runs as a single task to avoid needing shared locks between the two.
    """
    import time
    interval = 0.1  # 10 Hz

    while True:
        tick = time.monotonic()
        if registry.count > 0:
            await registry.broadcast(build_telemetry_envelope(state))
        await monitor.check(state)
        elapsed = time.monotonic() - tick
        await asyncio.sleep(max(0.0, interval - elapsed))

    Drains the MAVLink send queue and writes messages to the drone TCP connection.

    NOTE: pymavlink's blocking send methods are called in a thread pool executor
    to avoid blocking the event loop. In the current implementation, each message
    dict is serialised back to MAVLink bytes using the pymavlink mavutil connection
    that MavlinkReceiver holds internally.

    For initial integration, this stub logs outgoing commands. Full byte-level
    serialisation is completed when the drone-agent TCP send path is confirmed
    working in hardware testing.
    """
    while True:
        msg: dict[str, Any] = await queue.get()
        logger.info("MAVLink send: %s", msg)
        # TODO(hardware): serialise msg dict to MAVLink bytes and write to TCP stream.
        # Use mavutil.mavlink.MAVLink_<message_type>_message(...).pack(mavutil.mav)
        # then writer.write(packed) on the asyncio StreamWriter from receiver.run().
        queue.task_done()

@app.websocket("/ws")

    await websocket.accept()
    await _client_registry.add(websocket)

    # Send current connection status immediately on connect
    status_envelope = json.dumps({
        "type": "connection_status",
        "payload": {
            "drone_id": "DRONE-01",
            "status": "connected" if _telemetry_state and _telemetry_state.mode != "UNKNOWN" else "disconnected",
        },
    })
    await websocket.send_text(status_envelope)

    try:
        while True:
            raw = await websocket.receive_text()
            try:
                envelope = json.loads(raw)
                validate_ws_envelope(envelope)
            except (json.JSONDecodeError, SchemaValidationError) as exc:
                logger.warning("Malformed message from browser: %s", exc)
                continue

            msg_type = envelope["type"]
            payload  = envelope["payload"]

                except (SchemaValidationError, ValueError) as exc:
                    logger.warning("Invalid command: %s", exc)

                text       = payload.get("text", "")

                        "type": "copilot_response",

                        "payload": {
                            "response_type": "answer",

            else:
                logger.debug("Unhandled message type from browser: %s", msg_type)

        pass
```

## Design Decisions

**pymavlink asyncio integration:** pymavlink was designed for synchronous use. The receiver reads asyncio stream bytes in chunks and feeds them to pymavlink's internal buffer, calling `parse_char(b"")` to drain parsed frames. This keeps the event loop unblocked. The send path enqueues message dicts rather than calling pymavlink send methods directly — full byte serialisation is deferred to hardware testing (marked with a comment in `_drain_send_queue`).

**MAVLink send path stub:** `_drain_send_queue` in `main.py` logs outgoing commands but does not yet serialise them to bytes. This is intentional — the exact pymavlink send API call depends on whether drone-agent maintains a single persistent TCP connection or multiplexes. The queue-based design isolates command handling from this detail.

**WebRTC signaling drone WebSocket:** `SignalingRelay` is initialised with `drone_ws=None` in `main.py`. The production drone WebSocket connection (to `DRONE_SIGNAL_WS_URL`) needs to be established lazily in `lifespan`. This is left as a wiring step during hardware integration — the relay's `from_browser`/`from_drone` API is fully tested.

**Copilot model:** Defaults to `claude-sonnet-4-6` (current production Sonnet). Override via `COPILOT_MODEL` env var.

**Session history memory:** Copilot conversation history is stored in a Python dict — one entry per session UUID. This is suitable for a single-operator ground station. If multi-operator support is added later, history should move to a persistent store (Redis or SQLite).

**Mission monitor `_last_total` bootstrap:** The monitor sets `_last_total` from the first non-zero `mission_total` it sees. Tests that verify idle mode triggers must set `monitor._last_total` before calling `check()` — this reflects the real usage where the monitor has already observed a non-empty mission.
